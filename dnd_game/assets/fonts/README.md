# Kivy Fonts

The Kivy GUI looks here first for preferred font files.

- Story text: `Alegreya-Regular.ttf`, then Source Serif 4 variants.
- UI text: `Lato-Regular.ttf`, then Open Sans or Montserrat variants.
- Mono text: `JetBrainsMono-Regular.ttf`, then Inconsolata variants.

If these files are absent, the app falls back to close system fonts such as Georgia, Segoe UI, and Consolas.
